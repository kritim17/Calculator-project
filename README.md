# Simple Calculator (HTML, CSS & JavaScript)

This project is a basic yet visually modern calculator built using HTML, CSS, and JavaScript.  
It performs essential arithmetic operations like addition, subtraction, multiplication, and division, and also includes features such as percentage, negate, and keyboard input support.

This project was also used to demonstrate key Git and GitHub version control concepts such as:
- Initializing a repository
- Creating branches
- Committing changes
- Merging branches
- Creating and resolving an intentional merge conflict
